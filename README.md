# House Price Prediction

## Overview

This project implements Linear Regression from scratch using Gradient Descent and compares it with library-based methods. It also applies the Normal Equation and uses regression techniques for currency exchange rate prediction.

## Key Features

* Custom Linear Regression (Gradient Descent)
* Multivariate Regression
* Normal Equation implementation
* Comparison with Scikit-learn
* Time series prediction on currency data

## Technologies

Python, NumPy, Pandas, Matplotlib, Scikit-learn

## Datasets

* Boston House Prices (Kaggle)
* Currency Exchange Rates

## Results
* Successfully implemented regression from scratch
* Achieved comparable results with library models
* Observed faster execution using optimized libraries
* Generated meaningful predictions for housing and currency data
